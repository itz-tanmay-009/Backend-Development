# Backend Development Project

This repository contains backend development projects built using Node.js and Express.js.

## Projects

### 1. Todo Backend API
- Create Todo
- Update Todo
- Delete Todo
- Mark Todo as Read

### 2. Book Catalog API
- Add Books
- Update Books
- Delete Books
- Search Books by Title or Author

## Technologies Used
- Node.js
- Express.js
- REST API
- JavaScript
